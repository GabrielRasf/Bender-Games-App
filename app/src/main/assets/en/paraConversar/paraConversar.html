Game Rules

To Tak About
  • This game can be played in groups or as a couple.
  • One person will read a question and the other(s) will answer.
  • After answering, another person reads a new question and the cycle continues.
  • Variant:
  • If participants' answers match, the players who answered the same must drink a shot.
  • In this game, drinking is not mandatory; it is a lighter game aimed at getting to know each other with light and varied questions.
