Game Rules

×
  • One question, four choices, one correct answer
  • One player becomes the host and reads a question with 4 choices.
  • The next player in the circle answers.
  • If correct: turn passes.
  • If wrong: drinks a shot.
  • Then, the host role passes to the next player (clockwise).
  • VARIATION
  • The player can, once, ask for help from other participants
  • The player can, once, skip an answer
  • The player can, once, request to remove two options
  • Have fun! 🍻
