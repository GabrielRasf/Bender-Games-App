Eu Nunca

  • Cada jogador deve dizer "Eu nunca..." seguido de algo que nunca fez (ou que já tenha feito, caso queira beber).
  • Quem já fez o que foi dito, deve beber.
  • O objetivo é se divertir, compartilhar histórias engraçadas e conhecer melhor os outros participantes.

Topzeira

  • Uma pessoa lê uma proposta inicial, como "Alergias alimentares mais comuns no mundo".
  • Cada jogador, em sentido horário, dá uma resposta. Se alguém duvidar de uma resposta dita por qualquer participante, pode dizer "Não é topzeira".
  • Quando isso acontecer, ambos olham a resposta na "Resposta".
  • Se estiver na lista de Respostas, quem duvidou bebe; se não estiver, quem deu a resposta errada bebe.

Riu, Bebeu

  • Um jogador deve realizar uma ação proposta pela carta.
  • O objetivo é fazer o alvo rir em 30 segundos.
  • Se a pessoa rir, ela bebe! Se você não conseguir fazer o alvo rir, é você quem bebe.
  • Prepare-se para piadas, caretas e muita diversão!

Segredos

  • Confesse algo e tente enganar os outros. Fale uma verdade e uma mentira e deixe os outros descobrirem qual é qual.
  • Se os outros acertarem a verdade, você bebe uma dose.
  • Se errarem, o jogador ativo escolhe quem vai beber.
  • É uma maneira divertida de conhecer melhor os amigos e revelar segredos.

Piores Amigos

  • Quem é o "Pior Amigo" do grupo? Apontar o dedo para os outros acusando-os de atitudes bostas nunca foi tão engraçado!
  • Este jogo vai revelar muitas coisas divertidas sobre seus amigos.
  • Importante! Não leve para o coraçãozinho, é só um jogo.

Mímicas para Adultos

  • Faça mímicas ousadas e engraçadas para os outros adivinharem.
  • O jogo traz mímicas criativas e divertidas, ideais para quem gosta de desafios engraçados.
  • É uma chance de se soltar e criar momentos hilários!

Verdade ou Consequência

  • Escolha: responder sinceramente ou cumprir um desafio.
  • Se recusar ou mentir, você deve beber.

Quem no Grupo

  • Exemplo: "Quem no grupo é o mais fofo?" – todos apontam e quem receber mais votos deve beber!
  • É um jogo que revela opiniões do grupo de forma divertida e inesperada.

Para Conversar

  • Temas profundos, aleatórios ou engraçados para quebrar o gelo.
  • Ideal para iniciar conversas, conhecer melhor os outros ou se divertir com assuntos curiosos.

Acerta ou Bebe

  • Quem errar, deve beber. Simples e direto!
  • Este é um jogo rápido e direto, onde só existe uma regra: acertou, parabéns; errou, bebe!

O que é, o que é?

  • Nasce da zoeira, vive na resenha, faz rir, faz beber e ninguém reclama?
  • Bebedeira Games, ora essas... o que mais seria?
  • Se ninguém acertar a charada, todos bebem!

Perguntas sem Resposta

  • Exemplos: "Por que o céu é azul?", "Se o tempo é dinheiro, o banco é um relógio?".
  • O objetivo é levantar questões engraçadas ou filosóficas que ninguém pode responder com certeza.
  • Se alguém tentar responder e a galera achar fraca... bebe!

Intimidade

  • Perguntas profundas para descobrir segredos e histórias inusitadas.
  • Ideal para jogar entre casais, para aprofundar a relação com mais abertura e confiança.

Show do Bebão

  • Um quiz maluco com perguntas bizarras. Quem errar, bebe!
  • Prepare-se para dar risada e se surpreender com as perguntas.

Não fez? Bebeu

  • Desafios ou tarefas são propostos.
  • Quem não fizer, deve beber. Simples assim!

Tiro ao Alvo

  • 4 perguntas, uma resposta para cada pergunta.
  • A pessoa que fizer a pontuação mais alta, ganha
  • A pessoa que fizer a pontuação mais baixa, bebe!

Fato ou Falso

  • Você saberia dizer se o que você sabe é um fato ou é falso?
  • Neste jogo, você será apresentado a diversos conhecimentos
  • Se acertar o que é fato e o que é falso, parabéns.
  • Caso erre, já sabe, né!

Azar ou Sorte

  • Aqui você escolhe um, dois ou três números entre 1 e 6
  • Clica no botão e, se cair em algum número escolhido, já sabe. Bebe
  • Bebe uma dose
  • Jogo de pura sorte (ou azar)... risadas garantidas!

Antes ou Depois

  • Dois fatos históricos
  • Uma chance
  • Se errar, bebe
  • Se acertar, parabéns!
